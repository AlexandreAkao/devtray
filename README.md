## What it does

A menubar app. Hit `⌘⇧.` (default shortcut) to open a Spotlight-style launcher across 13 tools — type a few letters, pick a tool, paste, work, done. The app stays out of your way until you ask for it.

## Tools

- **JWT** decoder — paste a token, see the header/payload, verify signatures
- **JSON** formatter / minifier
- **Regex** tester with named captures
- **Base64** encode/decode
- **URL** encode/decode
- **Hash** (MD5/SHA-1/SHA-256/SHA-512)
- **UUID** generator
- **Timestamp** converter (Unix ↔ ISO, multiple timezones)
- **Snippets** manager with smart paste
- **Diff** viewer
- **Color** picker / converter (hex / rgb / hsl)
- **Cron** expression decoder
- **YAML** ↔ JSON

## Install

Download the latest DMG from the [Releases page](https://github.com/AlexandreAkao/devtray/releases/latest), open it, drag DevTray to Applications, and launch.

Requirements: macOS 14 (Sonoma) or later. Apple Silicon or Intel.
